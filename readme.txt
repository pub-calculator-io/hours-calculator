This online hour calculator is ideal for determining hours worked so that employers can determine pay accurately.

[https://www.calculator.io/hours-calculator/](https://www.calculator.io/hours-calculator/)

== Usage ==

Add the Hours Calculator shortcode to your page, post or sidebar:

`[ci_hours_calculator]`

Add the following code to your template where you would like the Hours Calculator to appear:

`<?php display_ci_hours_calculator(); ?>`

== Installation ==

1. Upload the Hours Calculator /ci_hours_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Hours Calculator plugin through the "Plugins" menu in WordPress.
